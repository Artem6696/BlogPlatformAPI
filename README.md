# BlogPlatformAPI



 Установка и запуск:

### 1. Клонируйте репозиторий:


   git clone [https://github.com/your-username/your-repository.git](https://github.com/Artem6696/BlogPlatformAPI.git)

### 2. Перейдите в каталог проекта:


  cd your-repository


### 3. Установите зависимости:


   npm install


### 4. Запустите проект:


   npm run start
